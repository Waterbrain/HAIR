## Asset Pipeline

When new brand assets are generated:

1. Generate using the locked master prompts in the brand guide (Sections 5 and 5.5)
2. Save the final selected render to this folder with the filename above
3. Update this README with a checkmark and any prompt deviations from the locked masters
4. If a render is rejected, do not save it here — keep prompt iterations in your own scratch folder until a final pick lands
